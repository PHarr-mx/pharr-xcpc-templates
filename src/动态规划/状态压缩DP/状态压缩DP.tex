\subsection{SOS DP}

SOS DP 又称高维前缀和，一般用来解决以下问题
\begin{framed}
对于所有的$i$, 满足$0\le i < 2^n$，求解$\sum_{j \in i} a_j$
\end{framed}
这里可以把$i$当做一个用二进制集合，每个物品只有选或不选两种情况。
